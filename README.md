# 🏥 Healthcare Data Analysis using Advanced Excel

## 📌 Project Overview

This project analyzes healthcare data using **Advanced Microsoft Excel** to uncover insights into patient demographics, optimize treatment pricing, and evaluate hospital resource utilization. The analysis leverages Excel's analytical and visualization tools to support informed decision-making and improve operational efficiency.

---

## 🎯 Objectives

* Analyze patient demographics and healthcare trends.
* Optimize treatment pricing based on patient and service data.
* Evaluate hospital resource utilization.
* Identify key performance indicators through interactive dashboards.
* Generate actionable insights using data visualization.

---

## 🛠️ Tools & Techniques Used

* Microsoft Excel (Advanced)
* Pivot Tables
* Pivot Charts
* Line, Bar & Pie Charts
* Excel Formulas & Functions
* Conditional Formatting
* Data Cleaning
* Data Validation
* Dashboard Design

---

## 📊 Key Analysis Performed

### 1. Patient Demographic Analysis

* Examined patient distribution by age, gender, and medical conditions.
* Identified demographic patterns and healthcare demand.

### 2. Patient Price Optimization

* Analyzed treatment costs across departments.
* Compared pricing trends to identify optimization opportunities.

### 3. Hospital Resource Analysis

* Evaluated utilization of hospital departments and available resources.
* Identified areas for improving operational efficiency.

### 4. Performance Dashboard

* Created interactive dashboards using Pivot Tables and Pivot Charts.
* Visualized key healthcare metrics for easy interpretation.
